# ActionRecognition
Action recognition semester project for Advanced Topics in Deep Learning: CSCE 689

### How to run demo

- `>>> cd demo`
- `>>> pip install -r requirements.txt`
- `>>> python demo.py` or `>>> python demo.py --vidpath ./videos/myvid.mp4`
